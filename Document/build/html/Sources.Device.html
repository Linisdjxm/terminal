ctlAddMark(Lat: float, Lng: float, popup: str = 'Mark', icon: str = 'cloud', color: str = 'blue')

在地图上添加标记点

参数:

  • Lat (float) – 经度

  • Lng (float) – 纬度

  • popup (str, optional) – 标记的命名. Defaults to “Mark”.

  • icon (str, optional) – 标记的图标. Defaults to “cloud”.

  • color (str, optional) – 标记的颜色. Defaults to “blue”.
